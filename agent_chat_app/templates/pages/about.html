About Agent Chat App

Learn more about our AI-powered chat application and how it can enhance your productivity.

What is Agent Chat App?

Agent Chat App is a modern, Django-based web application that combines the power of large language models with Retrieval-Augmented Generation (RAG) technology. This allows you to have intelligent conversations with AI models that can understand and reference your own documents and knowledge base.

Key Technologies

  • Django Framework - Robust web backend
  • Ollama Integration - Local LLM support
  • ChromaDB - Vector database for embeddings
  • WebSocket Support - Real-time chat interface
  • Bootstrap 5 - Modern responsive design
  • Document Processing - PDF, DOCX, XLSX, TXT
  • Dark Mode - Theme customization
  • User Management - Authentication & profiles

How RAG Works

1. Document Processing

Upload your documents which are then processed, chunked into manageable pieces, and converted into vector embeddings using the mxbai-embed-large model.

2. Vector Storage

These embeddings are stored in ChromaDB, a specialized vector database that enables fast similarity search and retrieval.

3. Query Processing

When you ask a question, it's converted to an embedding and similar document chunks are retrieved from the database.

4. Enhanced Response

The AI model uses both your question and the relevant document context to provide accurate, informed responses.
